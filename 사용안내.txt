경보이력 현황 PWA — 사용 안내
=================================

[구성 파일]
- index.html      앱 본체 (PDF 분석 + 대시보드)
- sw.js           서비스워커 (오프라인 캐싱)
- manifest.json   앱 정보 (이름/아이콘)
- icon-192.png / icon-512.png   앱 아이콘

[설치 방법]
1. 이 폴더의 파일들을 웹 서버 또는 GitHub Pages 등 https 주소에 올립니다.
   (※ PWA "홈 화면 추가"와 서비스워커는 https 또는 localhost에서만 동작)
2. 휴대폰 브라우저로 해당 주소에 접속합니다.
   - 아이폰(Safari): 공유 → "홈 화면에 추가"
   - 안드로이드(Chrome): 메뉴 → "앱 설치" 또는 "홈 화면에 추가"
3. 홈 화면 아이콘으로 실행하면 전체화면 앱처럼 동작합니다.

[사용법]
- "PDF 불러오기"를 눌러 경보이력 PDF를 선택하면 자동 분석됩니다.
- 통계 카드, 필터(정압기명/이벤트명/날짜/점검구분), 이벤트 목록이 채워집니다.
- 마지막으로 불러온 데이터는 저장되어 오프라인에서도 다시 열립니다.

[주의]
- PDF 표 형식이 샘플(ID 이름 이벤트 감지일시 복귀일시 경과시간 점검구분)과
  같은 구조여야 정확히 분석됩니다.
- 모든 분석은 휴대폰 안에서만 이루어지며 외부로 전송되지 않습니다.
