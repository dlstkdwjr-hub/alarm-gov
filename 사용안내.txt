경보이력 현황 PWA — 사용 안내 (v6)
===================================
[새 기능]
- 3개 PDF 데이터 51건 내장 (25년 1~2월 경주 / 25년 3~4월 외동 / 26년 업무외 경주)
- 앱에서 직접 이벤트 추가 / 수정 / 삭제 가능 (변경 내용 자동 저장)
- 필터 7종: 지역 / 정압기명 / 이벤트명 / 점검구분 / 날짜 / 시작시간대 / 종료시간대
- "기본(원본) 데이터로 초기화" : 내장 51건으로 복원

[데이터 입력/수정]
- "＋ 이벤트 추가" 버튼 -> 정보 입력 -> 저장
- 각 카드의 "수정" / "삭제" 버튼으로 편집
- 경과시간은 감지/복귀 일시로 자동 계산
- 모든 데이터는 휴대폰 안(localStorage)에만 저장됨

[설치]
1. 파일들을 GitHub Pages 등 https 주소에 올림
2. 휴대폰 브라우저로 접속 -> "홈 화면에 추가"
3. 업데이트 시 앱 삭제 후 재설치 권장
